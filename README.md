# pirte

etrip reversed. a small site of abstractions for the wandering mind — fractals, strange attractors, ascii, kaleidoscope, and terminal loops.

terminal ui, pure black, one accent, bento grid, nothing superfluous.

```bash
pnpm install
pnpm dev
```

## modules

- `/fractals` — mandelbrot + julia (webgl)
- `/attractors` — lorenz / clifford / dejong (canvas2d)
- `/ascii` — image → text
- `/terminal` — matrix rain · donut · life · flow · rule-30 (← → to cycle)
- `/pixels` — paint-by-number fill game
- `/time` — global clocks + session timer
- `/kaleidoscope` — n-fold mirror over a perlin field

`⌘K` to jump between modules.
